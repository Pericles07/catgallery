# 🐾 Cat Gallery

¡Bienvenido a **Cat Gallery**! Una aplicación web moderna y elegante diseñada para los amantes de los gatos. Explora una galería dinámica de gatitos con una interfaz limpia y una experiencia de usuario fluida.

---

## ✨ Características

* **Galería Dinámica:** Visualización de imágenes en tiempo real conectada a **The Cat API**.
* **Diseño "Mochi":** Interfaz minimalista con tonos cálidos y tipografía cuidada (**Quicksand** y **Montserrat**).
* **Responsive Design:** Cuadrícula adaptable para una visualización perfecta en cualquier dispositivo.
* **Interactividad:** Efectos de hover suaves en las tarjetas y botones de favoritos con estilo *glassmorphism*.
* **Carga Infinita:** Botón "Ver más gatos" para explorar contenido sin límites.

## 🌟 Sistema de Favoritos

Una de las piezas clave de **Cat Gallery** es la capacidad de interactuar con el contenido. Se ha implementado un sistema de favoritos que ofrece:

* **Persistencia Local:** Los gatos marcados se guardan en el `localStorage` del navegador, para que no pierdas tus elecciones al cerrar la sesión.

## 🛠️ Tecnologías utilizadas

* **Sass (SCSS):** Para un diseño modular y profesional con variables personalizadas.
* **JavaScript (ES6+):** Lógica asíncrona para la carga de datos y manipulación del DOM.
* **Vite:** Herramienta de desarrollo ultra rápida.
* **FontAwesome:** Iconografía para una interfaz intuitiva.
